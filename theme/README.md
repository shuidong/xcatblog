Theme for Xcat
===

A theme should at least have those files:

```nohighlight
theme
├── content.jsx
├── head.jsx
├── index.css
├── layout
│   ├── archive.jsx
│   ├── index.jsx
│   └── post.jsx
├── partial
│   ├── disqus.jsx
│   ├── footer.jsx
│   ├── google-analytics.jsx
│   └── header.jsx
└── scripts.jsx
```
